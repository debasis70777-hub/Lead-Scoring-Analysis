# Lead Scoring Analysis

## Project Overview
This project builds a Machine Learning model to identify potential customers who are most likely to convert into paying customers.

## Business Objective
The objective is to assign a lead score to each lead so that the sales team can prioritize high-conversion prospects.

## Dataset
- Leads.csv
- Leads Data Dictionary.xlsx

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Logistic Regression

## Project Workflow
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Model Building
5. Threshold Optimization
6. Model Evaluation

## Key Results
- Developed a Logistic Regression model for lead conversion prediction.
- Identified key factors influencing lead conversion.
- Generated lead scores to improve sales prioritization.
- Provided actionable business recommendations.
